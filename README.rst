LaTeX package for drawing graphical models and factor graphs with
TikZ.

Provides the following commands:

* \plate

* \shiftedplate

* \twofactor

* \factor

* \nofactor

* \namedgate

* \gate

* \vertgate

* \vertgateSpec

* \horgate

* \horogate

* \vertogate

Provides the following node styles:

* latent

* obs

* const

* factor
